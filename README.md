# AI 기반 대량전상자 전장 의료 후송 및 자원 최적화 플랫폼

국방 AI 활용 아이디어 경연대회 제출용 프론트엔드 단독 실행 데모입니다. 모든 부대, 병원, 환자, 위치 정보는 가상 데이터이며 실제 군사정보를 사용하지 않습니다.

## 이번 수정 반영 사항

- 환자 개별 좌표를 넓게 흩뿌리지 않고 사고 지점 기준 15~35px 반경의 환자집결지 클러스터로 밀집 배치했습니다.
- 지도에 `환자집결지 CCP / Casualty Collection Point` 마커를 추가하고, 주변에 작은 환자 점들이 모여 보이도록 했습니다.
- 사고 지점을 드래그하거나 더블클릭으로 이동하면 환자 클러스터도 함께 이동합니다.
- 병원 이동 시간, 병원 매칭, 후송 수단 추천은 환자 개별 좌표가 아니라 사고 지점/환자집결지 기준으로 계산합니다.
- 화면과 데이터 표시에서 `AI 위험 점수`를 제거하고, 분류, MARCH 플래그, 후송 우선순위, 추천 근거 중심으로 정리했습니다.
- 도로망을 주보급로, 보조로, 교량, 터널, 손상도로, 공사/차단 구간으로 세분화하고 지도에 도로명과 권장 경로를 표시했습니다.

## 실행 방법

```bash
npm install
npm run dev
```

빌드 확인:

```bash

```

## 주요 기능

- 시나리오 제어: 사고 유형, 환자 수, 기상, 시간대, 도로 상태, 적 위협도, 도로/교량 차단 이벤트
- 전술 지도: 산악 지형, 강, 교량, 도로망, 터널/협곡, 위험지역, 환자집결지, 후송 경로
- 환자 수정: 호흡수, 산소포화도, 혈압, 의식, 보행 가능 여부, 대량출혈, 흉부손상, 두부손상, CBRN 의심
- 즉시 재계산: 환자 수정 후 중증도 분류, MARCH 플래그, 병원 매칭, 후송 수단, 경로가 자동 갱신
- 병원 자원: 부대 의무대, 군병원, 화생방 전문치료병원, 국군수도병원별 병상/ICU/수술실/혈액/제독/격리 상태
- AI 종합 추천: 병목, 분산 후송 전략, CBRN 대응, 헬기 제한 여부, 국군수도병원 집중 방지 설명

## 발표용 시연 흐름

1. 기본 시나리오에서 A/B/C 부대 사이 사고 지점 주변에 환자집결지가 형성된 것을 보여줍니다.
2. 환자들이 넓게 흩어져 있지 않고 하나의 casualty cluster로 모여 있음을 설명합니다.
3. 지도에서 산, 강, 교량, 터널, 도로 차단 때문에 병원별 이동 시간이 달라지는 것을 보여줍니다.
4. 특정 즉각처치환자를 선택해 MARCH 플래그와 추천 병원/후송 수단을 확인합니다.
5. 혈압 저하, 호흡수 증가, 흉부손상 추가 등 환자 정보를 수정합니다.
6. 분류, 병원 매칭, 후송 수단, 경로가 즉시 재계산되는 것을 보여줍니다.
7. 도로 차단 또는 교량 차단 이벤트를 활성화해 권장 경로와 대체 경로가 바뀌는 것을 보여줍니다.
8. 최종적으로 제한된 의료자원을 분산 배정하고 의무요원·지휘관 판단을 보조하는 시스템이라는 메시지로 마무리합니다.

## Vercel 배포 절차

1. 이 폴더를 새 GitHub 저장소에 업로드합니다.
2. Vercel에서 `New Project`를 선택하고 해당 GitHub 저장소를 연결합니다.
3. Framework Preset은 `Vite`로 둡니다.
4. Build Command는 `npm run build`, Output Directory는 `dist`를 사용합니다.
5. 배포 후 발급되는 URL을 공유하면 접속할 수 있습니다.

### 링크 보유자만 접속하게 하는 방법

Vercel의 일반 배포 URL은 사실상 공개 URL입니다. 검색 노출을 막고 링크를 아는 사람만 접속하게 하는 수준은 가능하지만, 엄밀한 접근 제어가 필요하면 Vercel의 Deployment Protection을 켜야 합니다.

- Vercel 프로젝트의 `Settings` → `Deployment Protection`에서 Password Protection 또는 Vercel Authentication을 활성화합니다.
- Password Protection을 켜면 링크와 비밀번호를 가진 사람만 접속할 수 있습니다.
- 팀/요금제에 따라 기능 제공 범위가 다를 수 있습니다.

## 안전 고지

본 데모는 공개자료 기반의 가상 시뮬레이션이며, 실제 군 작전·의무 교범 또는 비공개 절차를 대체하지 않습니다. 환자 분류 결과는 실제 진료 판단을 대체하지 않으며, 현장 재평가가 필요합니다.
